# 포인터

### 포인터 변수 선언
```console
int* p;
```
띄어쓰기 주의!!

### 변수의 주소
```console
int main() {
	int a = 17;
	int *p;
	p = &a;
	cout << p << endl;
	cout << *p;
}
```
출력결과

000000649DDBF9C4

17

---

포인터는 배열처럼 쓸 수 있다.

```console
int a[9]
int *p=a;
```
a에 &을 붙이지 않는다.

배열은 포인터의 형식임.!!
